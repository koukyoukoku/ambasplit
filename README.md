# AmbaSplit

AmbaSplit is a desktop application for separate tracks (vocals, instrumental, etc.) using Demucs.

## Features

- Audio source separation powered by Demucs
- YouTube audio download using yt-dlp

---

## Install Guide

You must to install FFmpeg first in your computer, follow the instruction on this website to install:
https://www.gyan.dev/ffmpeg/builds/

Download the latest version from:
https://github.com/koukyoukoku/ambasplit/releases

1. Download the installer (.exe)
2. Run the installer
3. Launch AmbaSplit

---

## Development Requirements

If you want to run the project in development mode:

- Node.js 18+
- Python 3.11+
- FFmpeg (must be available in PATH)
